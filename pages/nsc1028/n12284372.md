 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      【大纪元2020年07月26日讯】现在新冠肺炎肆虐全世界。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      在瘟疫面前，人类除了被动的防控、医治之外，还能主动地做些什么，阻隔病毒的蔓延，守护我们的健康呢？
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      有这样一个故事：明朝有位名叫景清的读书人，他为人正直忠烈。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      他参加会试经过淳化的时候，借宿在当地的一户人家家里。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      这户人家的女儿，晚上常被妖怪折磨。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      而景清借宿的当晚，妖怪却没有来。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      景清离开后，妖怪就又来了。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      妖怪说是因为回避景秀才，所以才没来。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      女儿告诉了父亲，父亲追到景清处。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      景清书写了“景清在此”四字，让他们贴在门上。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      然后妖怪就不再骚扰了。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      这个故事说明：正邪不两立。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      比如：太阳升起，黑夜渐去。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      春暖花开，冰雪消融。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      邪不压正，正能驱邪，妖魔鬼怪是惧怕正人君子的。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      正人君子名号所在之处，魔鬼根本不敢兴妖作怪。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      同样，妖魔鬼怪在正法正神面前，更是望而生畏、避之不及的，因为正神是远远高于君子的。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      神能降妖除魔，道符能镇邪灭乱，真言能驱鬼祛病，在古代是人所共知的。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      现在越来越多的人，都知道了“
      <ok href="https://www.epochtimes.com/gb/tag/%E7%9C%9F%E5%96%84%E5%BF%8D%E5%A5%BD.html">
       真善忍好
      </ok>
      ，
     </span>
    </span>
   </span>
  </span>
  <ok href="http://www.minghui.org/mh/glossary.html#1">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       法轮大法
      </span>
     </span>
    </span>
   </span>
  </ok>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      好”
      <ok href="https://www.epochtimes.com/gb/tag/%E4%B9%9D%E5%AD%97%E7%9C%9F%E8%A8%80.html">
       九字真言
      </ok>
      ，可以驱邪扶正，能够化险为夷。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      很多人通过诚念
      <ok href="https://www.epochtimes.com/gb/tag/%E4%B9%9D%E5%AD%97%E7%9C%9F%E8%A8%80.html">
       九字真言
      </ok>
      ，有的疫鬼根本不敢靠近，有的在瘟病中不治而愈，有的在濒死时转危为安。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      他们的亲身经历，令人不可思议，给人信心希望。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      《念九字真言“法轮大法好、
      <ok href="https://www.epochtimes.com/gb/tag/%E7%9C%9F%E5%96%84%E5%BF%8D%E5%A5%BD.html">
       真善忍好
      </ok>
      ”—对武汉肺炎临床疗效的研究报告》，分析36名案例的总体结局是：26例症状痊愈，10例症状好转。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      3例病毒核酸检测由之前的阳性转为阴性。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      11例重症患者中有10例痊愈，1例好转。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      重症且住院的6例中有5例完全康复，1例好转。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      3例重症且住ICU，经医生诊断，2例完全康复，1例好转。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      其中一位居住在美国纽约长岛的珠宝商盖德（Osnot Gad），女，73岁，犹太籍美国人，报告者本人，报告日期2020年5月20日。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      报告如下：
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      “今年三月份，弟弟一家从曼哈顿来到长岛度周末。弟弟家离开一周以后，她发现自己嘴里除金属味之外，其它什么都尝不到。再一周后全身疼痛起不来床。医生诊断为武汉肺炎。盖德遵医嘱在家隔离，逐渐出现呼吸困难，无法呼吸，感到死亡的恐惧。四月下旬的一天，她的一个叫安娜的法轮功学员朋友来电话，得知她的情况之后，告诉她念“法轮大法好，真善忍好”九字真言。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      她发现，当她一个字一个字地念这句话的时候，“我的肺被打开了，让空气进入我的肺中，我能够把气吸得更长更深”，她说，“不管谁创造了这个口诀，这真的是让人正确呼吸的方法。”在慢慢地一声一声地重复九字真言的过程中，她喘过气来了，在接下来的三天中，盖德终于像正常人一样呼吸顺畅了。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      “我不知道这背后的原因是什么，这就是我的经历。”盖德说，她现在天天念诵“法轮大法好，真善忍好”。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      “不管是睡前，还是醒来的时候，还是在我走路的时候，还是我有不好的想法，或者抑郁的时候，我都念这个口诀。”
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      盖德发现，念九字真言不仅让她从武汉病毒中走了过来，而且她过去的心脏病以及其它病状都消失不见了。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      她一年前做过血管造影手术，她原来的心跳是每分钟44到46下，现在是50到54下。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      “我想说，这个口诀不是治了我一个病的问题，我有很多病，可这一切不好的症状都消失了。这个口诀对那些有呼吸问题的人，对血管有问题的人……对所有人，我敢说他对任何人都是有好处的，不只是对这个病毒患者，对任何人都好。”“我以前不相信神迹，但是我念九字真言确实全面改变了我的能量水平。”
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      九字真言，虽然只有九个字，但是汇集的是法轮佛法的正能量。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      所以念动真言之时，正神即刻就到，瘟鬼撒腿就跑。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      这就是念九字真言能治好新冠肺炎的根本原因。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      在中国传统文化中，神佛利用神通、法术降妖除魔的故事比比皆是，《封神榜》、《搜神记》、《西游记》、《阅微草堂笔记》等书籍中均有记载。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      鬼畏正神，神能驱鬼，道符有法力，真言有神威，曾是古代中国人普遍信服与依靠的。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      对于当前的新冠肺炎瘟疫，念动九字真言，瘟病不治而愈的案例越来越多。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      越来越多的人开始真心诚念九字真言，他们相信：九字真言胜于神丹妙药，没病可以预防，有病可以自愈，何乐而不为之呢？
     </span>
    </span>
   </span>
  </span>
 </p>
