 <p>
  【大纪元2019年09月19日讯】
 </p>
 <p>
  朋友，你从何处来，将向何处去？
  <br/>
  这你可曾想过？
  <br/>
  不要再沉睡，晨钟响彻了万里。
  <br/>
  你说“我要挣钱”，这不是你的错。
  <br/>
  但你要知道：
  <br/>
  这钱是否干净，还是滴着人血，背着罪恶？
  <br/>
  你说“我要做官”，这不是你的过。
  <br/>
  但你要明白：
  <br/>
  这官是否荣光，还是参与鬼祟，招致灾祸？
  <br/>
  做人，做鬼，上天正在看着。
  <br/>
  行善，行恶，上天都在记着。
  <br/>
  善有
  <ok href="https://www.epochtimes.com/gb/tag/%E5%96%84%E6%8A%A5.html">
   善报
  </ok>
  ，看谁积得善果。
  <br/>
  恶有恶报，有谁能够躲过？
  <br/>
  这不是空洞的说教，这是对世人的劝告：
  <br/>
  人生不过百年——
  <br/>
  天堂，地狱，你何处报到？
  <br/>
  这不是无事的生非，这是对生命的慈悲：
  <br/>
  人间倏忽一瞬——
  <br/>
  新生，毁灭，你何方相随？
  <br/>
  朋友，你要好好想一想！
 </p>
 <p>
  朋友，你为何而来，将怎样归去？
  <br/>
  这你可曾想过？
  <br/>
  不要再沉睡，曙光照亮了大地。
  <br/>
  大法是“真善忍”，中共是假恶暴。
  <br/>
  你选哪一个？
  <br/>
  不管是天地人，不管是你我他，谁逃得了？
  <br/>
  大法在救众生，中共在毁世人。
  <br/>
  你站哪一边？
  <br/>
  在正与邪面前，在生与死之间，谁脱得身？
  <br/>
  “九评”，“三退”，上天正在唤醒。
  <br/>
  天上，人间，上天都在裁定。
  <br/>
  除掉兽印，生命便是新生。
  <br/>
  摆脱附体，前途一片
  <ok href="https://www.epochtimes.com/gb/tag/%E5%85%89%E6%98%8E.html">
   光明
  </ok>
  。
  <br/>
  这不是空洞的说教，这是对世人的劝告：
  <br/>
  创世主到人间——
  <br/>
  救世，救人，佛光正普照。
  <br/>
  这不是无事的生非，这是对生命的慈悲：
  <br/>
  看人心选世人——
  <br/>
  谁留，谁去，全凭人自己。
  <br/>
  朋友，你要想想怎样做！
 </p>
 <p>
  注：“九评”，指《
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B9%9D%E8%AF%84%E5%85%B1%E4%BA%A7%E5%85%9A.html">
   九评共产党
  </ok>
  》；“三退”，指退党、退团、退队。
 </p>
