 <p>
  献给父亲的诗
 </p>
 <p>
  今天又是一个
  <ok href="https://www.epochtimes.com/gb/tag/%E7%88%B6%E4%BA%B2%E8%8A%82.html">
   父亲节
  </ok>
  <br/>
  却是我心尖的伤痛与隐患
  <br/>
  让我陷入久久的思念
  <br/>
  像风象雨又像雾伴随四季更迭
  <br/>
  父亲，你在哪里？
  <br/>
  逃离魔窟，妻离子散已经九年
  <br/>
  期盼，担忧，祈祷构成了每天的作业
  <br/>
  你的音容笑貌常在梦中浮现
  <br/>
  你的谆谆教诲让我后悔与你相伴的时间太短
  <br/>
  你沉默的背影和手指间的烟雾
  <br/>
  常让童年的我害怕你慈父的威严
  <br/>
  我忧伤地望着你的照片
  <br/>
  强忍悲伤，不敢流泪
  <br/>
  父女情深成了遥远的回忆
  <br/>
  我是你生命的延续
  <br/>
  我是你意志的传承
 </p>
 <p>
  我是你事业的接力手啊
  <br/>
  一言一行都要为你争得荣誉和骄傲
  <br/>
  关于你的传说有很多很多
  <br/>
  是生是死你早已超越
  <br/>
  你的生命来自黄土高原
  <br/>
  历经千万年的磨难而依旧屹立
  <br/>
  你的血液源自黄河之源
  <br/>
  穿越九曲十八弯浩浩荡荡奔向大海
  <br/>
  你被折磨被酷刑尝遍人间的苦难
  <br/>
  见证了魔鬼犯下的罪孽
  <br/>
  你被失踪被流放煎熬地狱里的岁月
  <br/>
  为人间匡扶公道和正义献出了自己的一切
  <br/>
  啊，父亲！
  <br/>
  我走过千山万水
  <br/>
  我到过异国他乡
  <br/>
  哪怕我站在天涯海角
  <br/>
  我要为你呼喊，为你歌唱
  <br/>
  我告诉鲜花，告诉飞鸟，告诉云彩
  <br/>
  我告诉大海，告诉蓝天，告诉地球那一边的星星和月亮
  <br/>
  让它们带去我对你深深的爱恋
  <br/>
  2018年6月17日
 </p>
