 <p>
  【大纪元2020年06月04日讯】（作者：秋阳）因着一位网友，我被带回了卅一年前那个对我而言淡得几乎为空白的时段。那些人、那段历史、铺展在我的面前，那样的生动鲜活，恍若置身其间，我的心与他们的热情、豪气、失望、恐惧、仓皇和绝望同消同长……
 </p>
 <p>
  三十一年，足以使一个意气风发、血气方刚的青年变得鬓发斑白、世故沧桑；三十一年，也足以让一段碾心蚀骨、撕裂肝肠的大恸结成疮疤，无关痛痒，只剩下哀伤后的静默和不动声色，以及忘却之后的心安理得。
 </p>
 <p>
  我看见你
  <br/>
  直面庞然的铁盔铁甲
  <br/>
  它们那样巨大 那样可怕
  <br/>
  似乎随时可以将你 碾碎 研磨成粉渣
  <br/>
  它们那样冷 那样硬
  <br/>
  似乎没有温暖可以将其融化
 </p>
 <p>
  我看见你
  <br/>
  面对魔兽的狰狞 舞爪张牙
  <br/>
  静静的伫立
  <br/>
  风
  <br/>
  也只吹动了你的衣领、裤脚和头发
 </p>
 <p>
  我看见你
  <br/>
  背影是那样的清瘦 那样的孤单
  <br/>
  你单薄而又挺直的脊梁
  <br/>
  却似承载了万钧重压
 </p>
 <p>
  我看见你
  <br/>
  赤手空拳
  <br/>
  没有利刃 没有宝刀
  <br/>
  只把那透亮的赤子心
  <br/>
  曝于坦克的炮口下
 </p>
 <p>
  然而
  <br/>
  我却看不见你
  <br/>
  那面对怪兽的年轻的面庞
  <br/>
  曾走过怎样的春秋冬夏
  <br/>
  我看不见你
  <br/>
  眼眸中是否闪烁着
  <br/>
  对爱情的憧憬，对母亲的眷恋，对家国命运的牵挂
 </p>
 <p>
  我看不见你
  <br/>
  是否对着铁兽喊话
  <br/>
  也许，你一直相信
  <br/>
  钢金铁甲包裹下的它们
  <br/>
  也一样热爱翱翔蓝天的飞鸟
  <br/>
  也爱阳光下的鲜花
 </p>
 <p>
  我看见的你
  <br/>
  离我那么的远
  <br/>
  远的看不清你的衣衫、你的容颜
  <br/>
  再见都无法辨识啊
  <br/>
  如何让我把你找还回家
 </p>
 <p>
  我看见的你
  <br/>
  距我又如此的近
  <br/>
  时隔31载 你依旧履约而来
  <br/>
  连我都嗅到了魔鬼嗜血的贪婪
  <br/>
  血盆大口要把千万个你吞下
  <br/>
  而你，却依然故我
  <br/>
  对千钧以一发
 </p>
 <p>
  我看见你
  <br/>
  你的灵魂 那么的孤寂却
  <br/>
  又那么的直立而宁静啊
  <br/>
  你胸口澎湃着的热度 点燃了那个初夏
 </p>
 <p>
  我看见你
  <br/>
  静默伫立，坚守了三十一载的光华
 </p>
 <p>
  可我，怎么却突然这样的怕啊
  <br/>
  怕你一转身 一回眸
  <br/>
  发现你身后的人们，
  <br/>
  早已鸟兽四散 顾左右而言他
 </p>
 <p>
  责任编辑：舜华
 </p>
