 <p>
  【大纪元2019年06月02日讯】谁能告诉我，今天是什么日子？
  <br/>
  我努力拨开历史的迷雾，从尘封的镜头窥探……
  <br/>
  银色的海滩，一群年轻人在海边搭起了帐篷，遥望着大海，敞开心中的理想。
  <br/>
  我们没有错！
  <br/>
  大海，我们今天集聚在海滩，只为能与你探讨未来前行的方向。
  <br/>
  大海在沉默，诡异般沉默的背后，是真相与谎言的较量、
  <ok href="https://www.epochtimes.com/gb/tag/%E8%87%AA%E7%94%B1.html">
   自由
  </ok>
  与枷锁的鏖战。
 </p>
 <p>
  曾记否？我们孜孜不倦的学习，我们执着地追求
  <ok href="https://www.epochtimes.com/gb/tag/%E4%BF%A1%E4%BB%B0.html">
   信仰
  </ok>
  ，渴望拥抱光明与太阳。
  <br/>
  戴上镣铐的思想，怎能为
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B0%91%E6%97%8F%E5%A4%8D%E5%85%B4.html">
   民族复兴
  </ok>
  而飞扬？
  <br/>
  检验真理的标准被禁锢在课堂。
  <br/>
  我们不要海市蜃楼，不要下跪的空谈，宁可站着独立思考，燃烧生命也无妨！
  <br/>
  而此刻，在广阔的海滩上，面对无情的大海，我们呐喊，我们呼唤，是如此悲壮！
 </p>
 <p>
  我们扎营在海滩上，我们露宿在海滩上，我们饥渴，我们流汗，在期待中守望。
  <br/>
  与我们对峙的大海，是如此的傲慢，貌似平静的海面下，是至高无上的龙王。
  <br/>
  不，
  <br/>
  这不是我要的大海，我要的大海是蔚蓝色的，她愿倾听我的诉求和期盼。
  <br/>
  难道这可怕的沉默后面，隐藏着什么？等待着我们的，是海的回音还是狂澜？
 </p>
 <p>
  拂晓时分，旌旗萧萧过红墙。远处传来轰隆的声音，在焖热的海滩上空回响。
  <br/>
  不对，这春夏的雷鸣声怎么有点异样？
  <br/>
  不会吧，大海真的要摧毁我们的志向？
  <br/>
  隆隆的声响越来越近，弥漫的硝烟骗过善良，疾飞的海燕穿破乌云，啊，来吧！
  <br/>
  让暴风雨来的更猛烈些吧！我们赤手空拳，我们无所畏惧，坚守海滩与
  <ok href="https://www.epochtimes.com/gb/tag/%E4%BF%A1%E4%BB%B0.html">
   信仰
  </ok>
  ！
 </p>
 <p>
  “年轻人，快快离开，海滩是我的地盘，清算的时刻到了！我是主宰大海的龙王”。
  <br/>
  深邃的海底，传来龙王声嘶力竭而又残酷的宣战！霎时，天昏地暗，海啸骇浪，龙王令伪装的八爪鱼，换上钢甲铁壳，冲向海滩，后面还有大群的虾兵蟹将。
  <br/>
  我们错了！错在太天真，太轻狂，就这样被碾碎了青春，被射穿了赤忱的胸膛。
 </p>
 <p>
  海啸过后，沉寂的海滩是那样的空旷，只有贝壳在哭泣，染红的衬衫在飘荡……
  <br/>
  初夏的夜空，被生命化成的流星照亮，为什么，为什么流星划过却更加黑暗？
  <br/>
  今夕是何年？
  <br/>
  当初坚守海滩的那群年轻人，在岁月静好声中“被”无情地遗忘。
  <br/>
  谁能告诉我，今天是什么日子？为何让苟活者失去重量，令王权者心惊胆战！
 </p>
