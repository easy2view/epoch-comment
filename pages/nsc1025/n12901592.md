 <p>
  【大纪元2021年04月24日讯】
 </p>
 <p>
  截止2021年4月19日，
  <br/>
  被誉为“中国良心”的
  <br/>
  大陆维权律师
  <ok href="https://www.epochtimes.com/gb/tag/%E9%AB%98%E6%99%BA%E6%99%9F.html">
   高智晟
  </ok>
  先生，
  <br/>
  已整整失踪3年多，1144天了。
  <br/>
  这3年多里，他没有电话，
  <br/>
  没有信息，没人知，什么都没有，
  <br/>
  人间蒸发了一般。
  <br/>
  直到这个月初，
  <br/>
  陕西榆林公安局才透露，
  <br/>
  人在他们手里，
  <br/>
  但不能会见，也不能通电话。
 </p>
 <p>
  算起来，高先生被关押迫害，
  <br/>
  迄今已长达十六年。
  <br/>
  这十六年里，
  <br/>
  他不是被酷刑，就是被单独监禁，
  <br/>
  并伴随多次被强制失踪，
  <br/>
  没有一天是自由的。
  <br/>
  但过去数次对他的绑架，
  <br/>
  都没最近这次这么长。
  <br/>
  这让他的妻子
  <ok href="https://www.epochtimes.com/gb/tag/%E8%80%BF%E5%92%8C.html">
   耿和
  </ok>
  女士，
  <br/>
  有一种越来越强烈的不详预感——
  <br/>
  <ok href="https://www.epochtimes.com/gb/tag/%E9%AB%98%E6%99%BA%E6%99%9F.html">
   高智晟
  </ok>
  很有可能已被迫害致死。
 </p>
 <p>
  4月19日，
  <ok href="https://www.epochtimes.com/gb/tag/%E8%80%BF%E5%92%8C.html">
   耿和
  </ok>
  在推特上发帖说，
  <br/>
  她曾幻想很快会回到中国，
  <br/>
  与丈夫共同生活；
  <br/>
  她也曾幻想高智晟刑满释放后，
  <br/>
  可以来美国弥补孩子缺失的父爱……
  <br/>
  “只是，我还是太天真了！
  <br/>
  我低估了中国共产党的邪恶，
  <br/>
  我现在什么梦想都没有，
  <br/>
  我只希望中国共产党
  <br/>
  可以从人道主义的角度出发，
  <br/>
  把高智晟的骨灰还给我，
  <br/>
  让我们一家团聚！”
 </p>
 <p>
  她还说，高智晟是因为
  <br/>
  太爱中国而受难的，
  <br/>
  他埋在祖国是他的荣光。
  <br/>
  如果这个卑微的要求
  <br/>
  都得不到满足，从今天起，
  <br/>
  她将把离家最近的中共国领土—
  <br/>
  中共国领事馆当成高智晟的墓地，
  <br/>
  “以后每逢月初、十五、清明节，
  <br/>
  我就带着孩子来这里祭拜，
  <br/>
  直到我死的那一天！”
 </p>
 <p>
  一个单薄弱小的
  <br/>
  普通中国家庭主妇，
  <br/>
  为了保护孩子不受株连，
  <br/>
  不得不远走异国他乡，
  <br/>
  不但无法与丈夫团聚，
  <br/>
  甚至连他在哪都不知道。
  <br/>
  在高智晟失踪的1144天里，
  <br/>
  耿和屡屡惊吓，夜不能眠……
  <br/>
  由最初的失望到最后的绝望，
  <br/>
  没有人知道，夜深人静时，
  <br/>
  她流过多少眼泪；
  <br/>
  更没有人知道，
  <br/>
  她经历了何种痛彻心扉的煎熬。
 </p>
 <p>
  高智晟夫妇的故事，
  <br/>
  其实是成千上万
  <br/>
  中国
  <ok href="https://www.epochtimes.com/gb/tag/%E8%89%AF%E5%BF%83%E7%8A%AF.html">
   良心犯
  </ok>
  家庭
  <br/>
  共同的故事。
  <br/>
  有多少高智晟这样的丈夫，
  <br/>
  就有多少耿和这样的妻子。
  <br/>
  她们的眼泪，她们的痛，
  <br/>
  是对
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1%E6%9A%B4%E6%94%BF.html">
   中共暴政
  </ok>
  的无声控诉！
 </p>
